===========================================
'Unity 디자이너스 바이블' 샘플 프로그램

Part 2/ Section 01 'Unity로 UI 화면을 만들어 보자!'
===========================================

■ 파일 설명
  - Unity_uGUI_Project_Start
    Unity로 UI화면을 만들 때 사용할 기본 데이터입니다.

  - Unity_uGUI_Project_Fin 
    Unity로 UI화면을 만든 완성 데이터입니다. 'Loading.unity' 씬을 열면 로딩이 시작되며, 시작화면 (SampleScene.unity)가 표시됩니다.

■ 샘플 파일 이용 시 주의점
  제공된 데이터는 책 학습용 샘플로 만들어진 것으로, 실제 사용을 보증하지 않습니다. 이 책의 독자가 학습용으로 사용하는 용도로만 허가합니다.
